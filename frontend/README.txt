1. 請實作下列需求
	Ans:
		請參考 index.html


2. 如果要加入第三個彩種，會怎麼進行擴充？
	Ans:
		在 Javascript 裡面加入第三個彩種的 option 設定


3. 如果要加入第四個玩法，會怎麼進行擴充？
	Ans:
		同彩種一樣，在 Javascript 增加第四個玩法的 option 設定
		並新增該規則檢查


4. 直選單式玩法中的刪除重複號，該如何改進效能？
	Ans:
		將每組數字串連成字串，然後用字串比對的方式去除重複號


5. 如果要設計 dark mode，該如何設計？
	Ans:
		設計開關，控制載入的 css 檔案
		或是 控制 class 來達到切換效果


6. 如果此專案有 i18n 的要求，該如何設計架構與翻譯等相關流程？
	Ans:
		撰寫一個 translator 物件，透過它來取得 i18n 內容
