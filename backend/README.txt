4. 每個號源有不同的速率限制，會如何實現限流，防止被 ban？
	Ans:
		將每個號源的速率限制設定在相對應的 class，在實作的時候參考自身的速率限制


5. 開獎時間並非準時，您會如何實現重試機制？

		2. 在後台增加手動 或 人工更新功能
